Piraeus Datastore Helm Charts

This is a Helm chart repository hosting Piraeus Operator charts.

Usage

Add the repository:

helm repo add piraeus-datastore https://kibamail.github.io/piraeus-datastore-helm-charts/

Update repositories:

helm repo update

Search for charts:

helm search repo piraeus-datastore

Install the Piraeus Operator:

helm install my-piraeus piraeus-datastore/piraeus --create-namespace -n piraeus-datastore

Available Charts

  • piraeus (v2.9.0) - The Piraeus Operator manages software defined storage clusters using LINSTOR in Kubernetes.

Repository Files

  • index.yaml - Helm repository index
  • piraeus-2.9.0.tgz - Piraeus chart package
